Instructions

The entire plugin is packaged into one file, including SVG and CSS. Versions:

  • Ticket machine line drawing: titoLoader.js
  • Tito word line drawing: titoLoader_word.js
  • Animated sliders: titoLoader_sliders.js

Then reference the JS file (after jQuery):


    <script src="javascripts/titoLoader.js"></script>

Note: Replace titoLoader.js with the filename if using the word or slider version

You can then apply the loader to the entire page by calling:


    titoLoader.show();

It can then be hidden using: 


    titoLoader.hide();

Partial page loading

The loader can be applied to specific containers on the page. To do this, pass a selector string:


    titoLoader.show('#mydiv');
    titoLoader.hide('#mydiv');
